# markdown 学习
## 第一章 **标题**
通常用“#”个数来表现标题层级。
一个#即为一级标题，以此类推。
要合理安排标题，不要越级使用标题（可参考书本目录类似排版）。
*注意：星号后必须有空格*

## 第二章 **段落**
大致上分为几种情况
### (1)直接回车
写完文字后直接回车，不可换段落，渲染出来后文字是连在一起的。
### (2)两个空格后回车
这是一个段落的第一行  
这是第二行（和上一行在同一个段落）
### (3)中间加一个空行
这是第一个段落

这是第二个段落
### (4)html换行标签
这是第一行。<br>
这是第二行。

## 第三章 **字体**
字体有四种：普通字体，*斜体*，**粗体**，***斜粗体***。

在这里，*等同于_，有相同效果，一般使用星号。

首先创建注脚[^注脚]。

## 第六章 **行内代码标记**
用处：区分代码和引用  
比如**粗体**这样使用是代码，`**粗体**`为引用。

## 第七章 **列表**
### (1)无序列表
无序列表使用星号(*)、加号(+)或是减号(-)作为列表标记，这些标记后面要添加一个空格，然后再填写内容。
推荐用-。

- 第一点
- 第二点
- 第三点
### (2)有序列表
用数字加.

1.
2.
3.
这种情况下，即使数字不连续也会自动修正。

1.
6.
4.
### (3)列表嵌套 
- 水果
  - 苹果
    - 红苹果
    - 绿苹果
  - 香蕉
  - 橙子
- 蔬菜
  - 胡萝卜
  - 白菜

这里的-和数字加.可交叉使用。缩进建议两个空格。

## 第八章 **引用块**
### 基础用法
Markdown 区块引用是在段落开头使用 > 符号 ，然后后面紧跟一个空格符号。

> 举个例子
>
>这就是一个引用块

> 也可以长引用
后面的几行
都可以不加`>`

和列表类似也可以嵌套

> 这是第一层
>> 这是第二层
>>> 这是第三层
引用块中可以使用大多数markdown元素（`##`、`*`一类的）
### 列表中用法
- 第一项
    > 引用块
- 第二项
    > 引用块

要加四个空格。

## 第九章 **代码**
### 行内代码
用`括起来，括起来的内容没有实际含义，仅表示引用。
### 代码区块
(1)缩进式代码块
正常文本段落

    这是缩进式代码块
    每行前面有四个空格，或者直接使用制表符Tab
    保持代码的原始格式
    
继续正常文本
*注意：代码块前后行需要空行隔离开。在列表中使用要8个空格。
(2)反引号代码块

```
多行代码内容
可以包含空行
保持原有缩进
```

在```后加语言标识可语法高亮

```javascript
const users = [
    { name: "Alice", age: 25 },
    { name: "Bob", age: 30 }
];

const adults = users.filter(user => user.age >= 18);
console.log(adults);
```
